PLY Viewer

Controls — Quick Hint

Mouse
  • Rotate: Left mouse drag
  • Zoom: Mouse wheel
  • Pan: Right mouse drag
Transform (Gizmo)
  • Click object to activate gizmo
  • T = Translate, R = Rotate, S = Scale
  • Esc = Detach gizmo
Labels & Selection
  • Ctrl + Click: add label at face centroid
  • Alt + Click: add label at nearest vertex
  • Double-click object → fit camera to it
  • Double-click label → change label text
